Methods

createMessage(expiringEmpresaItems) → {object}

Cria a mensagem que será enviada por e-mail. A mensagem é dividida em intro, details, tip e tipPath. intro contém o início da mensagem, com o conteúdo do que se trata; * details são os detalhes que vêm dos objetos gerados em getEmpresaExpiredItems; tip é a orientação para como resolver o alerta; * tipPath é o caminho que deve ser percorrido no sistema para resolver o alerta.

Parameters:
